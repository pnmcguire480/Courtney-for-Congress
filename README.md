# Cortney Peterson for Congress — Website Project

**Developer:** Patrick Nicholas McGuire  
**Client:** Cortney Peterson for Congress (Ohio's 11th District, Independent)  
**Client Role:** Cortney Peterson serves as both candidate and treasurer  
**Status:** Active

## Repository Contents

| File | Purpose |
|------|---------|
| `docs/website-agreement.html` | Interactive contract — fillable fields and electronic signature pads |
| `docs/website-agreement-signed.pdf` | Executed copy (added after both parties sign) |

## Agreement Summary

- **Project Fee:** $900 total, paid in $325/month installments
- **Maintenance:** $75/month after the project fee is fully paid
- **Cancellation:** Client may cancel at any time with no penalty

## Signing Process

1. Both parties open `docs/website-agreement.html` in a browser
2. Fill in contact info and effective date
3. Sign using the signature pad
4. Print to PDF → save as `docs/website-agreement-signed.pdf`
5. Commit the signed PDF to this repo for the permanent record

## Notes

- Electronic signatures are valid under the E-SIGN Act (15 U.S.C. § 7001) and UETA
- The contract is a self-contained HTML file with no external dependencies beyond Google Fonts
- Developer retains ownership of reusable tools/templates; campaign owns all campaign-specific deliverables
